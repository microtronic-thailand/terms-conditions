02 โมเดลการให้บริการรับจ้างและพัฒนาต่อยอด (Service and Customization Model) ผู้ใช้บริการรับทราบว่า ค่าบริการทั้งหมดเป็นค่าบริการเชิงวิชาชีพและค่าแรงงาน มิใช่สิทธิ์ผูกขาดในซอร์สโค้ดที่เป็น Open Source:

  • ค่าทักษะแรงงานและการปรับแต่งเฉพาะบุคคล (Customization Fee) ของผู้ให้บริการ
  • ค่าบริการจัดเตรียมสถาปัตยกรรมและนำส่ง (Deployment & Setup Fee) ขึ้นระบบโฮสติ้งและโดเมน
  • ค่าสนับสนุนการปฏิบัติงานทางเทคนิค (Technical Support Fee) การชำระเงินจึงเป็นไปเพื่อแลกเวลา ทักษะแรงงาน และการประกอบสร้างระบบเท่านั้น

03 กรรมสิทธิ์ในทรัพย์สินทางปัญญาและการอ้างอิง Open Source

  • ซอร์สโค้ดโครงสร้างและเฟรมเวิร์กอย่าง Next.js หรือ Vite อยู่ภายใต้ใบอนุญาตสากลแบบเปิด (MIT License) ผู้ใช้บริการไม่มีสิทธิ์อ้างกรรมสิทธิ์เดี่ยวในส่วนแกนหลัก Open Source
  • ผู้ใช้บริการจะได้รับสิทธิ์อันสมบูรณ์ในงานออกแบบ ภาพลักษณ์ แบรนด์ และโค้ดส่วนขยายเฉพาะทางธุรกิจ (Custom Proprietary Logic) ที่จัดทำขึ้นใหม่เพื่อผู้ใช้บริการโดยเฉพาะ

04 การจำกัดความรับผิดและขอบเขตความเสียหาย ระบบถูกส่งมอบ "ตามสภาพที่เป็นอยู่จริง" (As-Is) ผู้ให้บริการไม่ต้องรับผิดชอบต่อความสูญหายของข้อมูลหรือความเสียหายทางธุรกิจอันเกิดจากความบกพร่องของซอฟต์แวร์ต้นทาง (เช่น Node.js, Next.js หรือ Third-party Modules) ผู้ใช้บริการตกลงตรวจสอบและทดลองระบบให้มั่นใจก่อนประยุกต์ใช้งานเชิงพาณิชย์จริง
